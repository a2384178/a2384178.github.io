我的第3篇文章

1.新建：$ mkdir pracitcegit
$ cd practicegit
$ pwd//显示路径

2.初始化：$ git init//变成仓库

3.添加到仓库：$ git add gitpractice.txt

4.提交到仓库：$ git commit -m "first submit"

5.$ git status //查看状态

6.$ git diff//查看和上修改的不同

7.$ git log //查看历史记录

8.$ git reset --hard HEAD^//回退到上一个版本，

9.$ git reset --hard 36525656//回退到数字指定的版本。数字为commit的ID，可以不用写全

10.$ git reflog//会记录每一次的命令

11.$ git checkout -- file //丢弃工作区的修改。让这个文件回到最近一次git commit或git add时的状态。

12.$ git rm file//删除文件，之后记得commit

13.如果删错了，还可以$ git checkout用版本库的替换工作区的

克隆：

14.$ git clone git@github.com:a2384178/gitskills.git//要把仓库换成自己的名字

分支管理：

15.$ git checkout -b dev//-b表示创建分支dev并且切换到dev上

16.$ git branch 查看当前分支，*表示当前的

对dev分支修改commit后切回master

17.$ git checkout master

18.$ git merge dev//把dev上面的工作成果合并到master上面。

小结：查看分支：git branch

创建分支：git branch

切换分支：git checkout

创建+切换分支：git checkout -b

合并某分支到当前分支：git merge

删除分支：git branch -d
